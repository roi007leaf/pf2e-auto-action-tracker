# PF2e Automated Action Tracker

A built in tracker for counting the number of actions taken by each combatant.  Useful for new players and GMs alike - never wonder again how many actions
a combatant has left, or if a particular combatant has used their reaction.  This module is designed to handle all of that for you - so you can focus on
the fun parts of combat - **destroying your enemies!**

### Features
* **Automatic Pip Tracking**: Injects action pips directly into the Combat Tracker for real-time tracking.
* **Movement Intelligence**: Detects all move actions. It even calculates cost adjustments for Difficult Terrain and Greater Difficult Terrain using V12 Regions.
* **PF2e Rule Integration:**
    * **Quickened** Handles Gold pips based on conditions present at the start of the turn.
    * **Slowed/Stunned**: Automatically drains actions at turn start and decrements the Stunned condition per RAW.
    * **Reactions**: Tracks reaction usage per combatant.
* **Complex Action Handling**: Support for complex activities with subordinate actions.
    * **Subordinate Actions:** Handles logic for things like Doctor's Visitation, tracking the Stride followed by your choice of healing skill (with varied cost)
    * **Interrupted Movement:** Supports advanced feats like Dual-Weapon Blitz. You can pause your Stride, resolve Strikes, and resume moving; the module tracks your total coordinate history against the system ruler to ensure you stay within your movement budget.
    * **Graceful Degradation:** If an activity is "broken" (e.g., moving too far during a complex action), the tracker automatically converts the movement to a standard action and notifies the user and GM via whisper (if set to receive these types of whispers).
    * **Predictive Hover**: Forgot what step of a complex action you're on? Hover over the pips to see what’s expected next!
* **Smart Detection**: Distinguishes between a player simply linking a spell or item in chat versus actually clicking Cast/Use.
* **Undo & Reroll Protection**: Automatically refunds actions if a chat message is deleted or a movement is undone (Ctrl+Z). Corrects the log when system rerolls are used.
* **Sustain Tools**: Sends reminders with clickable buttons to Sustain a spell (logging an action) or let it lapse (with partial cleanup).
* **Privacy & Stealth**: Respects Secret Rolls; players will not see pips generated by rolls made hidden to the GM.

### Interactive Controls
* **Hover:** Get detailed info about exactly what each pip represents.
* **Left-Click:** Jump directly to the chat card that generated the action.
* **Right-Click:** Manually remove a pip (perfect for accidental clicks).
* **Manual Button:** A dedicated button to add actions for unique edge cases.


## Installation
Requires the following modules:
* [lib-wrapper](https://foundryvtt.com/packages/libWrapper)
* [socketlib](https://foundryvtt.com/packages/socketlib)

## Settings
* **Whisper Alerts:** Configure alerts for overspending, underspending (ending turns with actions left), reminders to Sustain spells (default on), breaking complex actions (using movement in the middle of flurry of blows), and undoing (via Ctrl + Z) forcing removal of other actions.
* **Debug Mode:** Verbose logging for troubleshooting complex action triggers.
